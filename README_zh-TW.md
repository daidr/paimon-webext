# 派蒙 Paimon

<img align="right" src="./screenshots/popup.png" width="320px" height="auto" alt="screenshot_popup" />

> 在瀏覽器中即時查看原神遊戲狀態

特色功能:

* 🌙 顯示即時樹脂數值及恢復情況
* 💰 探索派遣/每日委託/洞天寶錢/參數質變儀/周本 Boss 提示
* 🌏 完整的國服(含世界樹) & 海外服支援
* 🚨 樹脂/參數質變儀等閾值通知
* 📚 多帳號支援

## 開發

```bash
pnpm i

pnpm dev
```

## 調試

### firefox:

```bash
pnpm start:firefox
```

### chromium:

```bash
pnpm start:chromium
```

## 構建

```bash
pnpm build
```
